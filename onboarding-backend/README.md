# Backend

## Development

1. Install: `yarn install`
2. Start Docker: `docker-compose up -d`
3. Generate Prisma Schema: `npx prisma generate`
4. Migration: `npx prisma migrate dev`
5. Start in Dev Mode: `yarn start:dev`
